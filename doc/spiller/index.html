SpiBasic

This is a useful "generic" spiller object. Both SpiObject and SpiGroup extend this class, as do the plugins. Has no size, position or graphical data.

SpiCamera

The camera class is used to display the game's visuals in the Flash player.
By default one camera is created automatically, that is the same size as the Flash player.
You can add more cameras or even replace the main camera using utilities in SpiG.

v1.0 Initial version

SpiG

SpiObject

This is the base class for most of the display objects
(SpiSprite, SpiText, etc).
It includes some basic attributes about game objects,
including retro-style flickering, basic state information,
sizes, scrolling, and basic physics and motion.

v1.0 Initial version
